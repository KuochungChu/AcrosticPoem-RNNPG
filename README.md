# 安裝方法 

請參照[RNNPG] (https://github.com/XingxingZhang/rnnpg)

並安裝python簡繁轉換：[https://pypi.python.org/pypi/Jianfan](https://pypi.python.org/pypi/Jianfan)

# 使用方法

## 1.
至 keyword.txt 輸入想要藏頭的字（限四字），字與字之間要空一格，例如：

```
電 腦 賦 詩
``` 

## 2.
執行以下指令：

```
sh run.sh
```

## 3.
程式會自動產生以下結果：

```
電光明月夕陽紅
腦子清風草色濃
賦得不知南行處
詩來盡是種不容
```
